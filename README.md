## 二.技術棧

（react18）React Hooks

（react庫）React Router6，redux，react-lazyload，redux-thunk

（Http請求交互）axios

（UI庫）react-transition-group，react-weui，antd

（構建工具）Vite2

（打包工具）rollup


## 三.項目亮點

1.請求封裝

2.實現搜索功能 + 防抖（debounce）

3.頁面以及圖片懶加載（lazy-load）

4.redux數據管理

5.輪播圖二次封裝（swiper）
